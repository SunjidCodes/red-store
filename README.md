
# Red Store | Modern E-commerce UI

RedStore is a fully responsive, multi-page e-commerce front-end application designed for a modern shopping experience. This project showcases a clean, minimalist UI for a fashion brand, featuring product catalogs, detailed item views, a shopping cart, and user account management.



## Features

- **Responsive design:**  Optimized for all screen sizes (mobile, tablet, and desktop) using CSS Flexbox and Grid.

- **Multi-Page Navigation:** Seamless transitions between the Home, Product, Product Details, Cart, and Account pages.

- **Dynamic UI Elements:** Interactive product sliders, menu toggles for mobile view, and hover effects.

- **Product Organization:** Featured products and latest items categorized for easy browsing.

- **Shopping Cart UI:** A dedicated layout for managing selected items before checkout.

## Technologies Used

- **HTML5:** Semantic structure for the web pages.
- **CSS3:** Custom styling including Google Fonts integration and FontAwesome icons.
- **JavaScript:** Logic for the mobile navigation menu and image galleries.
